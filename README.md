# 🏠 대한민국 아파트 실거래가 대시보드

국토교통부 공공데이터포털 API를 활용하여 **지역별 아파트 매매 실거래가**를
주식 시세처럼 시각화하는 파이썬 웹 대시보드입니다.

---

## 📸 주요 기능

| 기능 | 설명 |
|------|------|
| 🗺️ **인터랙티브 지도** | 전국 지도에서 지역 마커 클릭 → 즉시 상세 정보 표시 |
| 💰 **거래가 분포** | 선택 지역의 거래금액 히스토그램 |
| 📈 **월별 추이** | 최근 N개월 평균 거래가 시계열 차트 |
| 📋 **최근 거래 목록** | 아파트명 · 면적 · 층수 · 거래일 상세 목록 |
| 🔄 **자동 갱신** | 30분마다 자동으로 최신 데이터 갱신 |

---

## ⚡ 빠른 시작

### 1. API 키 발급 (무료, 자동승인)

1. [공공데이터포털](https://www.data.go.kr) 접속 → 회원가입
2. 검색창에 **"국토교통부 아파트 매매 실거래가"** 검색
3. **활용신청** 클릭 → 목적 간단히 작성 후 신청
4. **마이페이지 → API신청** 에서 서비스키 확인 (보통 1~2시간 내 자동승인)

### 2. 설치

```bash
pip install -r requirements.txt
```

### 3. API 키 설정

**방법 A: 환경변수 (권장)**
```bash
# macOS/Linux
export MOLIT_API_KEY="발급받은서비스키"

# Windows PowerShell
$env:MOLIT_API_KEY="발급받은서비스키"
```

**방법 B: 코드 직접 수정**
```python
# korea_apt_dashboard.py 상단에서
API_KEY = "발급받은서비스키"   # ← 여기에 직접 입력
```

### 4. 실행

```bash
python korea_apt_dashboard.py
```

브라우저에서 **http://localhost:8050** 접속 🎉

## 🔧 API 정보

- **출처**: 국토교통부 아파트 매매 실거래가 자료
- **데이터**: 계약 기준 월별 거래 내역
- **업데이트**: 신고 즉시 (통상 계약 후 30일 이내 신고)
- **무료 쿼터**: 일일 1,000건 (개발계정 기준)

---

## 📌 참고사항

- 국토교통부 실거래가는 **계약일 기준** 데이터이며, 신고 지연으로 최신 데이터가 늦게 반영될 수 있습니다.
- 한 달치 데이터를 전 지역 조회 시 API 호출이 많아질 수 있으니 지역을 선택하여 조회하세요.
- API 키 없이 실행 시 데이터 없음 메시지가 표시됩니다 (UI 미리보기는 가능).
